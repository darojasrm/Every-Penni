# Money Tracker App

A full-stack expense tracking application built with React, Node.js, Express, and PostgreSQL.

## Features

- User authentication with JWT
- Add, edit, delete expenses
- Category system
- Expense filters
- Dashboard stats
- Line chart and pie chart analytics

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- Recharts

### Backend

- Node.js
- Express
- PostgreSQL
- JWT Authentication

## Project Structure

- `frontend/` – React app
- `backend/` – Express API

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```
